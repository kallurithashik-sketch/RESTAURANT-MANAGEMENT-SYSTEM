Restaurant Management System in C
Project Description

The Restaurant Management System is a simple console-based application written in C language. It allows users to:

Display restaurant menu
Add customer orders
Generate and view bill
Save bill into a text file (bill.txt)
Exit the program

This project uses:

Structures
Arrays
Functions
File Handling
Switch Case
Menu Items
ID	Item Name	Price
1	Andhra Fry Piece Biryani	Rs 250
2	Mutton Biryani	Rs 420
3	Chicken Manchurian	Rs 200
4	Paneer Fried Rice	Rs 180
5	Lemon Mojito	Rs 120
How to Run
Compile
gcc restaurant.c -o restaurant
Execute
./restaurant
